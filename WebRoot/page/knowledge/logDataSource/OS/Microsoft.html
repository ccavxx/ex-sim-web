1.1 Windows 
通过安装代理可以采集 windows 的应用日志、系统日志和安全日志。采集日志之前，请打开 windows 的审核策略修改其安全设置（打开方式：运行gpedit.msc进入本地组策略编辑器>计算机配置>Windows设置>安全设置>本地策略>审核策略，按照下图2.1所示，进行策略的修改），如此可以使日志内容更详细。

运行services.msc，进入系统服务，找到名称为Windows Management Instrumentation服务，开启该服务，如下图：

若主机不允许开启WMI服务或者安全管理系统安装在linux系统上无法使用WMI服务，则必须通过在被收集主机上安装安全管理系统代理来收集Windows日志，步骤如下：

在安全管理系统平台上进入产品支持->产品信息，点击“Agent下载”，安装Agent并开启Agent服务即可。
