5.6 北电
名称 北电交换机 类别 网络产品
日志源配置 设备类型 交换机-> 北电-> Passport
北电交换机配置 
可以使用串口线直接连接，如果交换机配置有管理IP并且允许telnet登陆，则也可以通过telnet进行登陆。
1、	输入enable命令进入特权模式。
2、	输入config进行配置模式。
3、配置syslog日志接收主机地址
sys syslog host 1 create  
sys syslog host 1 address 10.2.1.100 
sys syslog host 1 host enable  
sys syslog host 1 severity info warning error fa安全管理系统
