12.1.1 瑞星防毒墙
名称 瑞星防毒墙 类别 安全产品
日志源配置 设备类型 反病毒 -> 瑞星 -> 瑞星防毒墙
瑞星防毒墙WEB页面配置 系统菜单 日志审计 -> 日志配置 -> 日志设置
备注 勾选远程日志配置中的“syslog”，“主机IP”填入安全管理系统的IP地址，端口填入“514”。
12.1.2 瑞星杀毒软件网络版
名称 瑞星杀毒软件网络版 类别 安全产品
日志源配置 设备类型 反病毒 -> 瑞星 -> 瑞星杀毒软件网络版
瑞星杀毒软件配置 系统菜单

1.进入瑞星杀毒服务器，双击打开“管理控制台”。

2.点击【工具】【当前系统中心设置】，弹出配置对话框。

3.选择“报警插件设置”—“瑞星SNMP陷阱报警插件”，点击“添加报警”，进入设置页面。

4.填入描述，可任意输入报警插件名称；报警内容默认选择“病毒日志”，点击“下一步”。

5.接下来直接点击“下一步”，在如下图所示的“SNMP设置”，依次输入日志服务器的IP地址、端口号为162，团体名为public，点击“下一步”。

6.勾选“报告已有日志”，点击“完成”即可。
